# Project Context: Alpha Cyg (Hugo Site)

- **Framework**: Hugo (Static Site Generator).
- **Theme**: PaperMod.
- **Content**: Personal blog/portfolio with posts, photos, and an about page.
- **Development Tasks**:

    - Customize PaperMod theme features via `hugo.toml`.
    - Handle static assets and potentially custom CSS/JS.
- **Instructions**:
    - When suggesting changes to the site, prioritize `hugo.toml` configuration and standard Hugo directory structures.
    - Provide Hugo-specific commands (e.g., `hugo server -D`).
